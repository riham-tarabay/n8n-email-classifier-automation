## Prerequisites
- n8n instance (cloud at n8n.cloud or self-hosted)
- OpenAI API key (platform.openai.com)
- Google account for Sheets/Gmail OAuth2

## Import Steps
1. Open n8n → top right menu → **Import workflow**
2. Upload the `.json` file from this folder
3. Add credentials when prompted
4. Update any hardcoded IDs (Sheet IDs, emails)
5. Test with **Execute workflow** before activating
